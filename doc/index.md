# Install

## Dependeincies

```
sudo apt install python-serial python-pip qt4-designer qt4-dev-tools pyqt4-dev-tools
sudo pip install minimalmodbus
```

## Install Procedure

```
python setup.py build
python setup.py install --prefix="${HOME}"
```
